# CloudStorage云存储

#### Description
CloudStorage是一款基于Java语言实现的轻量级文件管理和分享平台，使用Spring Boot和Vue3构建。它支持用户注册、登录、上传/下载文件、创建/修改/删除文件夹，以及简单的文件分享和权限管理功能。
